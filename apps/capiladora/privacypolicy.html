Privacy Policy

This Privacy Policy describes how the calculator application ("we", "our", or "the App") handles user information.

1. Data Collection and Usage

The App stores only two types of data:

  • Operation history: To allow the user to view previous calculations.
  • Theme preferences: For interface customization.

This data is stored locally on the user's device and is not shared with third parties.

2. Permissions

The App does not require special permissions to function. All data remains within the application itself.

3. Data Sharing

We do not share any user information with third parties.

4. Security

Since the data is stored locally, it is subject to the security of the user's own device. We recommend keeping the device secure to protect this information.

5. Changes to This Privacy Policy

We may update this policy periodically. Users will be notified if there are significant changes.

6. Contact

If you have any questions about this policy, please contact us at mrcapybarastudios@gmail.com.
