Política de Privacidade

Esta Política de Privacidade descreve como o aplicativo de calculadora ("nós", "nosso" ou "o App") lida com as informações dos usuários.

1. Coleta e Uso de Dados

O App armazena apenas dois tipos de dados:

  • Histórico de operações: Para permitir que o usuário visualize cálculos anteriores.
  • Preferências de tema: Para personalização da interface.

Esses dados são armazenados localmente no dispositivo do usuário e não são compartilhados com terceiros.

2. Permissões

O App não requer permissões especiais para funcionar. Todos os dados permanecem dentro do próprio aplicativo.

3. Compartilhamento de Dados

Não compartilhamos nenhuma informação do usuário com terceiros.

4. Segurança

Como os dados são armazenados localmente, eles estão sujeitos à segurança do próprio dispositivo do usuário. Recomendamos manter o dispositivo seguro para proteger essas informações.

5. Alterações nesta Política de Privacidade

Podemos atualizar esta política periodicamente. Notificaremos os usuários caso haja mudanças significativas.

6. Contato
